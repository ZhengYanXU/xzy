نُشر في 2023-07-07

飞书笔记转Markdown

因为一直使用飞书维持自己的知识库且飞书在文档编辑上相当顺手，因此一些笔记也就直接使用飞书编辑。

但是飞书无法直接导出为Markdown格式。

好在飞书可以很方便的将笔记导出为word，然后使用pandoc工具便可将word转化为Markdown格式发布到博客上。

Step1：安装pandoc

官网下载链接：https://www.pandoc.org/installing.html

Step2：验证是否安装成功

输入命令：pandoc --version


Step3：转换

使用命令：

pandoc -f docx -t markdown test.docx -o test.md --extract-media ./images
